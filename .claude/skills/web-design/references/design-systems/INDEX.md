# Design Systems Index

58 个真实网站的设计系统预设，来源于 [VoltAgent/awesome-design-md](https://github.com/VoltAgent/awesome-design-md)（MIT License）。

## 使用方式

**按需读取，不要全量加载。** 当用户提到"做一个类似 XXX 的页面"时，读取对应文件获取完整设计规范。

每个文件包含 9 个标准化章节：
1. Visual Theme & Atmosphere — 设计哲学和氛围
2. Color Palette & Roles — 完整色彩系统
3. Typography Rules — 字体层级表
4. Component Stylings — 按钮/卡片/输入框/导航样式
5. Layout Principles — 间距/网格/容器规范
6. Depth & Elevation — 阴影体系
7. Do's and Don'ts — 设计护栏
8. Responsive Behavior — 响应式断点
9. Agent Prompt Guide — 快速参考和组件示例
